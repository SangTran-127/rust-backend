* languages: Rust, postgresSQL

 * Opentelemetry: Tracing
 * gRPC: tonic/protobuf

# Installation
 * Cargo make:
 ```
 cargo install cargo-make
 ```

 * Cargo Nextest:
 ```
 cargo install cargo-nextest
 ```

 * Cargo Watch(like nodemon in JS)
 ```
 cargo install cargo-watch
 ```

 * Build
 ```
 cargo make build
 ```
